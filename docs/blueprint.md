# **App Name**: Ecos Portfolio

## Core Features:

- Project Showcase: Display past XV Años dance projects with photos and videos.
- Interactive Choreography Catalog: Browse and preview available dance choreography styles with short video previews.
- AI Style Recommendation: Generate suggested choreography styles based on user input like theme and music using a generative AI tool.
- Personalized Quote Request: Allow users to request personalized quotes for specific dance packages and choreography options.
- Contact Form: Enable users to directly contact Ecos Del Sur with questions or booking inquiries.

## Style Guidelines:

- Primary color: Use the deep red from the logo (#A7202C). Reflects passion and the deep 'Ecos Del Sur' company name.
- Secondary Color: Use the gold from the logo (#B3914F) to create a visually appealing interface.
- Background color: Soft Beige (#F5F5DC). Provides a neutral, earthy backdrop, making content and imagery stand out.
- Accent color: Muted Gold (#B8860B). This offers a touch of elegance and sophistication.
- Headline font: 'Playfair', a modern serif with a high-end feel.
- Body font: 'PT Sans', a humanist sans-serif for body text.
- Use culturally relevant, elegant icons for categories and interactive elements.
- Subtle transitions and animations when browsing projects.
